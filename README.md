# Pilot Finance

Pilot Finance is a mobile application developed to simplify financial and operational management for smartphone repair businesses. The application helps streamline daily business processes by providing automatic service price calculation, customer management, warranty tracking, voucher management, and comprehensive financial reporting.

## Features

- Automatic Service Price Calculation
- Customer Management
- Device Management
- Warranty Management
- Coupon & Voucher Management
- Voucher Redemption
- Financial Transaction Management
- Income & Expense Recording
- Profit Calculation
- Total Balance Monitoring
- Export Financial Reports

## Tech Stack

- Flutter
- Dart
- Firebase

The application was built to improve operational efficiency by reducing manual calculations and organizing business data in a centralized mobile platform.
